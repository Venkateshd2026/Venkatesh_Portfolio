# Certificate files

Copy your certificate files from **My Documents** (or any folder) into this folder using the **exact** filenames below. The portfolio links each "View Certificate" button to these files.

| Certificate | Filename |
|-------------|----------|
| Brains & Bytes-25 Hackathon | Hackathon-Certificate.jpg |
| DevOps and its Applications | Devops_certificate.jpg |
| Internet of Things (IoT) Workshop | iot_workshop_certifcate.jpg |
| HTML & CSS Bootcamp | htmlbootcamp.pdf |
| Java Programming – Beginner to Master | java udemy.pdf |
| Java Bootcamp | javabootcamp.pdf |
| SQL Bootcamp | sql_boot _camp.pdf |

- **Images** (.jpg, .png) and **PDFs** (.pdf) are both supported.
- Filenames are case-sensitive and must match exactly (including spaces in `java udemy.pdf` and `sql_boot _camp.pdf`).

After copying, "View Certificate" will open the file in the modal viewer.
